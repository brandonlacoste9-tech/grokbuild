# The Grok Collective

Official digital home of **The Grok Collective** — a multi-agent building team (Grok, Benjamin, Harper, Lucas). Premium cosmic UI, interactive agent cards, process visualization, showcase, and a “Brief the Collective” form with simulated multi-agent responses.

Built with **Next.js (App Router)**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**. Custom agent portraits via Grok Imagine.

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Development server |
| `npm run build` | Production build |
| `npm run start` | Serve production build |
| `npm run lint` | ESLint |

## Project structure

```
src/
  app/           # App Router entry, layout, global styles
  components/    # Hero, Agents, Process, Showcase, Brief, Footer
  lib/           # Agent data, brief response simulation
public/agents/   # Imagine-generated portraits
docs/            # Product brief
```
